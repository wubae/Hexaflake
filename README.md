# Hexaflake

This is a C++ project that uses [SFML](https://www.sfml-dev.org/) (Simple and Fast Multimedia Library) to render graphics, handle input, and play sounds.

---

## Requirements

- **C++ compiler** (e.g., `g++`, `clang++`)
- **SFML 2.5.1+**
